# Git GUI — Installation & Lancement

## Prérequis
- Node.js v18+ ([télécharger](https://nodejs.org))
- Git installé sur votre machine

## Installation (une seule fois)

Ouvrez un terminal dans ce dossier et lancez :

```bash
npm install
```

## Lancement

```bash
npm run dev
```

L'application s'ouvre automatiquement.

## Fonctionnalités

- **Graphe de commits** — Visualisation des branches avec lignes colorées (style GitKraken)
- **Opérations Git** — Fetch, Push, Pull depuis la toolbar
- **Gestion des branches** — Checkout, création, suppression (clic droit dans la sidebar)
- **Diff de fichiers** — Vue inline avec additions/suppressions colorées
- **Recherche** — Filtrage des commits par message, auteur ou SHA
- **Toutes branches** — Basculer entre la branche courante et toutes les branches

## Raccourcis

| Action | Méthode |
|--------|---------|
| Ouvrir un dépôt | Bouton en haut à gauche |
| Voir le diff d'un commit | Cliquer sur un commit |
| Checkout une branche | Double-clic dans la sidebar |
| Menu contextuel branche | Clic droit dans la sidebar |
| Rafraîchir | Bouton ↻ dans la toolbar |
